# 🎬 高级影视清单管理系统

一个功能强大、界面精美的个人电影观影清单管理工具。

## ✨ 主要功能

### 🎯 核心功能
- **电影搜索与添加**：通过 TMDB API 搜索电影，支持智能搜索建议
- **多语言支持**：中文、英文、双语显示切换
- **预告片播放**：直接在应用内观看 YouTube 预告片
- **详细信息展示**：电影海报、评分、简介、上映日期等

### 📝 新增功能（v2.0）

#### 1. **笔记系统**
- 为每部电影添加个人观影笔记
- 支持长文本记录
- 笔记标记显示

#### 2. **个人评分**
- 0-10分评分系统
- 独立于TMDB评分
- 滑块式评分界面

#### 3. **观看状态管理**
- 🔵 **想看**：计划观看的电影
- 🟠 **在看**：正在观看中
- 🟢 **已看**：已完成观看
- 一键切换状态
- 自动记录观看时间

#### 4. **智能筛选**
- 按观看状态筛选
- 实时更新显示

#### 5. **多维度排序**
- 最新添加
- 最新/最早年代
- TMDB评分最高
- 个人评分最高

#### 6. **统计面板**
- 总电影数量
- 各状态电影数量
- 平均TMDB评分
- 平均个人评分
- 实时数据更新

#### 7. **数据管理**
- 📤 **导出数据**：导出为JSON格式
- 📥 **导入数据**：从JSON文件导入
- 🗑️ **清空数据**：一键清空所有记录
- 💾 **自动保存**：LocalStorage持久化

#### 8. **用户体验优化**
- ⌨️ **键盘快捷键**：
  - `Ctrl + K`：聚焦搜索框
  - `Ctrl + E`：导出数据
  - `ESC`：关闭弹窗
- 🔔 **通知系统**：操作反馈提示
- ⏳ **加载状态**：搜索时显示加载动画
- 📭 **空状态设计**：清单为空时的友好提示
- 🖼️ **图片懒加载**：优化性能

#### 9. **视觉增强**
- 🎨 **9种主题**：Netflix、Midnight、Royal、Forest、Ocean、Light、Soft、Sky、Rose
- 🔤 **6种字体**：Inter、Poppins、Playfair、Space Mono、Outfit、Fredoka
- 🌈 **透明度调节**：自定义UI透明度
- ✨ **精美动画**：入场动画、过渡效果、悬停效果
- 📱 **响应式设计**：完美适配移动端

#### 10. **设置面板**
- 集中管理所有设置
- 透明度调节
- 数据管理
- 快捷键说明

## 🚀 使用方法

### 基本操作
1. **添加电影**：在搜索框输入电影名称，选择建议或直接搜索
2. **查看详情**：点击电影卡片查看完整信息
3. **添加笔记**：点击📝按钮添加观影笔记
4. **评分**：点击⭐按钮为电影评分
5. **更改状态**：点击状态按钮切换想看/在看/已看
6. **删除电影**：点击✕按钮删除电影

### 高级功能
- **筛选**：使用顶部筛选按钮按状态查看
- **排序**：使用排序按钮重新排列电影
- **主题切换**：点击🎨选择喜欢的主题
- **字体切换**：点击✍️选择字体风格
- **语言切换**：点击🌐切换显示语言
- **导出数据**：设置面板 → 导出数据
- **导入数据**：设置面板 → 导入数据

## 🛠️ 技术栈

- **前端**：原生 HTML + CSS + JavaScript
- **API**：TMDB (The Movie Database)
- **存储**：LocalStorage
- **字体**：Google Fonts
- **视频**：YouTube Embed

## 📊 数据结构

每部电影包含以下信息：
```javascript
{
  id: Number,              // TMDB电影ID
  titleCn: String,         // 中文标题
  titleEn: String,         // 英文标题
  poster: String,          // 海报URL
  backdrop: String,        // 背景图URL
  rating: Number,          // TMDB评分
  releaseDate: String,     // 上映日期
  releaseYear: String,     // 上映年份
  overviewCn: String,      // 中文简介
  overviewEn: String,      // 英文简介
  trailerKey: String,      // YouTube预告片KEY
  userRating: Number,      // 个人评分 (0-10)
  watchStatus: String,     // 观看状态 (unwatched/watching/watched)
  notes: String,           // 个人笔记
  tags: Array,             // 标签（预留）
  addedDate: String,       // 添加时间
  watchedDate: String      // 观看时间
}
```
